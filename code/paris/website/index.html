Data Series Exploration

Fast interactive similarity search for data series exploration requires low query answering times, a problem traditionally attacked through indexing. Such indexes allow for large parts of the raw data to be pruned.

The problem is that they take hours or even days to be built. This can be a show-stopper for many applications that either require immediate access to the data or the amount of queries does not justify the cost that has to be paid upfront.

We develop systems that manifest the benefits of adaptive indexing for interactive exploration of large data series collections. The intuition is that instead of building database indexes up-front, indexes are built during query processing, adapting to the workload. We allow users to explore large multi-gigabyte datasets without waiting for days, pose exact or approximate similarity queries using the mouse or touch screen and get their answers in seconds.

We have developed the ADS (Adaptive Data Series) family of indexes and RINSE (RINSE Interactive Series Explorer), a tool for interactive exploration of large data series collections.
